nand2tetris

Posted on: Sat 21 January 2017

Tags: #mooc, #coursera

Nand to Tetris is an interesting book/teaching aid that I had heard of before, but couldn't quite find the time to pick up. A few months ago, a teaser was uploaded announcing that material from this book would be adapted into a Coursera MOOC. This piqued my interest. I would be following along this MOOC in a self paced manner and uploading code to my personal repo as I go along.

I would starting with Unit 0 : Introduction and Unit 1 : Boolean Functions & Logic Gates today. I would update the repo with the code as I progress along.
